article

A tag article e uma seção ( parte) do site que nai precisa do site todo para fazer sentido, ou seja, é uma região auto-explicativa.

O aria-labelledby é um atributo utilizado quando se quer evitar repetições de palavras proximas no leitor de tela, ex: se você colocar um titulo 'celular' e uma descrição aria-label = 'celular', o leitor de tela lerá duas vezes a palavra celular causando desconforto. Então, utilizamos im ID no titulo e um ara-labelledby com, assim o leitor de tela lerá somente uma vez.

main

A tag main guarda o conteudo principal do site

Tag OL, UL, DL e LI

A tag UL é utilizada para criar listas não ordenadas ( unordered list). Ex:

  • Menus de um site
  • Lista de tópicos mais visitados
  • Lista de ingredientes

Ol é uma lista ORDENADA (ordered list). Ex:

Top 5 produtos mais vendidos do mês:

  1. Barbeador Eletrico
  2. Fritadeira S Oleo
  3. Carrinho de Controle Remoto
  4. Mouse DRagon Fight 3mil
  5. Banco imobiliario

Tag section

A tag section pode ser utiizada para agrupar o conteudo do site em diferentes regioes. Geralmente o conteudo necessita do seu contexto para ser compreendido.

semanticas para usar o texto

Blockquote
Utilizado para fazer citaçõe no HTML. Possui um atributo chamado 'cite', onde colocamos nele o valor: o local onde retiramos a citação
Q
Marca uma citação inline, ou seja, dentro do texto.
Cite
Tag que serve para marcar obras (livros, filmes, pinturas, etc)
em
Tag utiliada para dar enfase em uma parte do texto
strong
utilizado para marcar uma parte importante do site, a deixa em negrito

A tag aside é utilizada para marcar conteudo complementar ao principal do site.

Exemplos: Lista de posts mais visitados, informações estras como a descrição de um termo usado no artigo e mais.

Tag Header

Marca o cabeçalho do site (banner), onde geralmente estão presentes a marca, navegação do site e as vezes uma informação introdutoria.

Tag Footer

É utilizada no final do site, geralmente possui dados de contato, outros inks, infos de copyright.
